# calc - gt
This is a demonstration and an example on how to integrate and use gt in a C projet. The project used in this example is called **calc** : a small program that calculates basic statistics from a list of numbers entered from the command line. 

## Setup
Run the script `setup.sh` to download and install gt. 

## Build 
Run the script `build.sh` to build the project.

## Run 
To run the program, write `./out/program test`. This will run the tests written for the `calc` project.
